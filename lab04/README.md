
1. What does the Expo default template add to the minimal blank template (cf. lab 1)?

   This template adds the essential boilerplate code and libraries needed to build the app.

2. How is the styling of the app widgets specified (cf. CSS)?

   The styling is specified using StyleSheet objects in React Native.

3. What does Expo store in the app/ sub-directory?

   Expo stores the main application code (which includes the index and layout files in the default template, then the about and not-found files).

4. What does Expo store in the components/ sub-directory?

    Expo stores the Button and ImageViewer component code in the components/ sub-directory.

5. When we `npx expo install` the expo-image-picker, what files are changed in the application?
    The package.json files are changed to include the new dependencies.
